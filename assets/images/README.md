# Images

Place your image assets here. Suggested:

- `jennifer-portrait.jpg` — hero portrait (~1200×1500px, 4:5 ratio)
- `jennifer-about.jpg` — about section editorial portrait (~900×1200px, 3:4 ratio)
- `og-image.jpg` — Open Graph share image (1200×630px)
- `favicon.svg` — site favicon

Then update the image references in `../../index.html` (uncomment the `<img>` tags) and remove the placeholder gradients in `../../css/styles.css` (the `::after` pseudo-elements on `.hero-image-frame` and `.about-image-frame`).
